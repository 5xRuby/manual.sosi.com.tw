## 使用者資訊與清單

### 搜尋與篩選

管理員可透過輸入使用者關鍵字串，快速篩選出相關的帳號結果。

### 清單項目說明

可檢閱使用者 ID、認證方式（如 LDAP 或 SOSI）、使用者帳號、名稱、身分（使用者/管理員）、2FA 啟用狀態及最後登入時間。

![使用者清單](/images/screenshots/zh/image034.jpg)

### 批次匯入

支援透過上傳 .xlsx 或 .csv 檔案的方式，批次建立使用者資料。

### 查看紀錄

- **使用者資訊：** 可查看設定資訊。
- **操作紀錄：** 包含使用者執行的事件動作、來源 IP 地址、更新時間、項目類型及項目編號等資訊。

![使用者紀錄](/images/screenshots/zh/image036.jpg)

## 帳號維護操作

**建立與編輯：** 管理員可手動新增使用者，或進入編輯頁面修改帳號、名稱、電子信箱、身分權限及時區設定。

![帳號編輯](/images/screenshots/zh/image038.jpg)

**帳號刪除：** 管理員可將不再需要的使用者帳號從系統中移除。

## 兩階段驗證碼啟用

### 開啟設定

> 此路徑為針對單一使用者，可至全站設定開啟 All User

**重置 2FA 按鈕：** 若該使用者已經啟用二階段認證，但需要重新設定兩階段認證（例如遺失），管理員可以協助點選此按鈕，讓使用者登入後重新設定兩階段認證。

![2FA 設定](/images/screenshots/zh/image040.jpg)
