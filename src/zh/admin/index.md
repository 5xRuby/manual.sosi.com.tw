## 快速設定

管理員初次使用 SOSI 時，請依以下四步驟完成基本設定：

### 1. 建立使用者

**目的：** 建立能登入 SOSI 系統的操作人員。

**動作：** 於「使用者列表」點擊「建立使用者」，設定其登入系統的帳號與密碼。

### 2. 建立裝置

**目的：** 定義要連線的遠端主機（如 Windows 或 Linux）。

**動作：** 於「裝置列表」新增裝置，填入 IP 地址並選擇正確的類型（RDP、SSH 或 VNC）。

### 3. 建立憑證

**目的：** 儲存遠端主機的帳號密碼（這就是自動登入的「鑰匙」）。

**動作：** 於「憑證設定」建立憑證，輸入該主機真實有效的帳號與密碼。

### 4. 授權連線

**目的：** 將「人、門、鑰匙」綁定在一起。

**動作：** 進入「授權裝置一覽」，點擊「建立裝置連線授權」，並在授權憑證欄位選取步驟 3 建立的憑證。

## 功能總覽

管理員控制台提供完整的系統管理功能，包含裝置管理、使用者管理、憑證設定、連線授權、浮水印防護、領域認證、全站設定及安全審計等模組。

| 功能模組 | 說明 |
|---------|------|
| [裝置管理](/zh/admin/devices/) | 管理遠端連線裝置（新增、查看、編輯、設定） |
| [使用者管理](/zh/admin/users/) | 管理使用者帳號（建立、編輯、2FA 設定） |
| [憑證設定](/zh/admin/credentials/) | 管理遠端主機登入憑證 |
| [裝置連線授權](/zh/admin/connection-auth/) | 管理授權群組與連線權限 |
| [浮水印設定](/zh/admin/watermark/) | 設定螢幕浮水印防護機制 |
| [領域設定](/zh/admin/domains/) | 管理 LDAP 與認證領域 |
| [全站設定](/zh/admin/site-settings/) | 2FA 與郵件全域設定 |
| [安全審計](/zh/admin/audit/) | 連線紀錄、檔案傳輸與操作追蹤 |
